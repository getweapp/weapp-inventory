# 微信小程序－抽屉列表

### 说明：

实现抽屉列表效果，使用weui.wxss库。

### 数据接口:

使用本地数据

### 目录结构：

- images — 存放项目图片文件
- pages — 存放项目页面文件
- style — 存放weui.wxss文件
- utils — 存放格式化文件

### 开发环境：
